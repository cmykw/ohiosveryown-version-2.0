Overview

Concepts is a Cambridge, Massachusetts based retailer with various locations throughout the northeast United States. They specialize in the footwear and sportswear markets. I was approached to redesign their website with responsive capabilities. This was a great project for me to work on because I definitely consider myself a sneakerhead, and Concepts is a very popular retailer in that regard. So combining two things I know well, design and sneakers, I set forth to create an awesome experience for prospective buyers.

Problem: Currently, the Concepts website is not responsive and this was a big driving force in the redesign. We wanted to create an optimized mobile experience not only for browsing and reading blog entries, but also for purchasing goods. So with that in mind, a clear, easy to use purchase path was crucial. When browsing the current site from a desktop or laptop screen, a lot of the main calls to action (images, buttons, hyperlinks) are inconsistent in regards to styling and position. It was important to unify a solution in order to create a sense of location and direction for the new site.

Solution: I touched on a few key solutions above, but to summarize, we wanted to be sure to not only design and account for mobile devices, but to do so in a way where the experience is optimized and no compromises are made as far as content (or any other pieces of information). We also wanted to give the prospective buyer direction and awareness on the redesigned site, but we wanted to do that by evolving the current design of the site. That meant clean open spaces where images and typography drive the sites content. We wanted to make the redesign edgy, so I did things like creating “off-center” buttons and headers. All together, I was very pleased with how the redesign turned out. Check it out below.
