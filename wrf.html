Winthrop Rockefeller
Foundation Website Redesign.

Contributions: Responsive Web Design, User Experience, User Interface,
Information Architecture, Iconography.

Overview

For nearly 40 years, the Winthrop Rockefeller Foundation has provided funding for projects across Arkansas to encourage economic development, education, and racial and social justice. I was approached to redesign the website, create a visual style guide, and create various iconography and visuals.

Problem: The previous website was outdated visually, but more damaging was the disorganization and inconvenient user experience. The site has important information for not only residents of the state of Arkansas, but also for a larger majority of people in general. An unclear site navigation and disorganized content flow is damaging and off-putting, even when a site's content is important, such is the case with the Winthrop Rockefeller Foundation.

Solution: It was my goal to not only create a great visual experience on the surface, but to also clearly present the vital information the site currently houses. This was possible by creating a clear communion of both information and visual appearance, complimenting and supporting each other.
